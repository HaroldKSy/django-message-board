# django-message-board


## Description
A simple Django message board app where users can post and reply to messages.
Built to practice full-stack development with Django, Git, and deployment.

## Features
- Create Posts
- Reply to messages
- Clean UI

## Tech Stack
- Python
- Django
- SQlite
- HTML/CSS
- Git & GitHub

## Project Structure
```
project/
 | -- app/
 | -- templates/
 | -- static/
 | -- manage.py
```

## Installation / Setup

```
1. Clone the repo:
git clone https://github.com/yourusername/project-name.git
cd project-name

2. Create virtual environment:
python -m venv venv
source venv/Scripts/activate

3. Install dependencies:
pip install -r requirements.txt

4. Run migrations:
python manage.py migrate

5. Start server:
python manage.py runserver

## 6. usage
Explain how to use it:
- Go to http://127.0.0.1:8000/
- Create a post
- Reply to others
```
